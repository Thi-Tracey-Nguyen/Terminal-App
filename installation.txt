1. Download src folder to your Desktop

2. Open terminal by: 
	- On Mac: pressing Command + space, type Terminal, hit enter
	- On window: pressing Windows key + X then click Command prompt, at the command prompt, type: bash then hit Enter

3. In the terminal, change the working directory to Desktop

4. Open the downloaded file: by typing 'open src' in the terminal

5. Allow the execution of the bash script: 'chmod +x word_game.sh'

6. Run the program: './word_game.sh'
